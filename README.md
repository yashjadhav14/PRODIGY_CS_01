## 📖 Project Overview
The Caesar Cipher is one of the simplest encryption techniques. It works by shifting each letter in a message by a fixed number of positions in the alphabet.

---

## ✨ Features
- Encrypt text using a shift value
- Decrypt encrypted text
- Supports uppercase and lowercase letters
- Ignores special characters and numbers
